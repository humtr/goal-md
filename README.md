# goal-md

`goal-md` is a small seed repository for goal-oriented Markdown notes and lightweight planning artifacts.

## Status

Active seed repository. The current repository is intentionally minimal and should either grow into a documented Markdown workflow or be archived if it is no longer needed.

## Purpose

Use this repository to keep simple goal documents, planning notes, and Markdown templates under version control.

## Usage

1. Add goal documents as Markdown files.
2. Keep filenames short and descriptive.
3. Prefer plain Markdown over tool-specific formats.
4. Review and prune obsolete goals regularly.

## Validation

There is no build step yet. For now, validation is manual:

- Check Markdown files render correctly on GitHub.
- Keep links relative when possible.
- Avoid committing private credentials, tokens, or personal secrets.

## Next steps

- Add a `templates/` directory if repeated goal formats emerge.
- Add a short example goal document.
- Decide whether this repository should remain active, be merged into another notes repository, or be archived.
